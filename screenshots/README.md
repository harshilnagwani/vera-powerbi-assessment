## How to Take Screenshots

In Power BI Desktop:
1. Navigate to the page you want to capture
2. Press `Windows + Shift + S` (Windows Snipping Tool)
3. Save as PNG to this folder

Then link them in README.md using:
```markdown
![User Intelligence](screenshots/user_intelligence.png)
```
