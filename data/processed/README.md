# Processed Data

This folder contains cleaned datasets ready for Power BI import.
Generated by the pre-processing scripts — do not edit manually.

## Files Generated

| File | Generated By | Description |
|------|-------------|-------------|
| `Delivery_Logistics_Clean.csv` | `scripts/fix_operations_timestamps.py` | Fixed timestamps + engineered columns |
| `HR_Clean.csv` | `scripts/clean_hr_dataset.py` | HR with engineered columns |

## How to Regenerate

```bash
python scripts/fix_operations_timestamps.py
python scripts/clean_hr_dataset.py
```

Both scripts print a QA summary so you can verify output before importing to Power BI.
